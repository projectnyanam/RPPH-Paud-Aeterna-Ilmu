# Petunjuk Kolaborasi AI

Aplikasi ini adalah alat bantu guru TK di Indonesia. Ikuti aturan berikut:

1. **Bahasa**: Gunakan Bahasa Indonesia yang edukatif, ramah, dan profesional di semua UI dan konten yang dihasilkan AI.
2. **Desain**: Pertahankan estetika "Ceria" dengan warna pastel yang lembut tapi kontras, sudut membulat (rounded-3xl), dan animasi halus.
3. **Typography**: Gunakan font Serif untuk judul (font-serif) untuk kesan elegan dan Sans untuk UI fungsional.
4. **Keamanan**: Kunci API Gemini tidak boleh diekspos ke client. Gunakan endpoint `/api/generate-rpph`.
5. **Responsivitas**: Selalu gunakan utility Tailwind untuk memastikan UI berjalan baik di mobile (guru sering menggunakan HP saat di kelas).
